# agent-swarm

Maquinário de agentes multi-runtime extraído de projetos reais: skills, lints determinísticos,
hooks e contratos de operação (CLAUDE.md / AGENTS.md).

- `codex/` — trilha OpenAI Codex (conteúdo original deste repositório): contratos de handoff de
  loops, schemas, testes de regressão e verificação executável.
- `claude/` — trilha Claude Code: padronização e curadoria de documentação — wiki Karpathy
  temporal, `docs-wiki-lint`, `ref-integrity` git-aware (pre-commit + skill + loop),
  self-improvement loop (lessons.md) e CLAUDE.md com workflow orchestration de Boris Cherny.
  Mapa completo em `claude/README.md`.
